# UI/UX Prompts

## Responsive Refinement Prompt

- **Services:** `contact360.io/app`, `contact360.io/root`, `contact360.io/email`, `extension/contact360`
- **Prompt:** Make pages fully responsive for mobile/tablet/desktop with consistent spacing and accessibility contrast.
- **Outputs:** updated layouts, responsive verification checklist
- **Small Tasks:** audit breakpoints -> update layouts -> test viewport matrix

## Theme Consistency Prompt

- **Services:** frontend apps and extension
- **Prompt:** Apply and validate dark/light theme consistency across shared components and pages.
- **Outputs:** unified theme variables and visual checks
- **Small Tasks:** token map -> component updates -> visual QA
