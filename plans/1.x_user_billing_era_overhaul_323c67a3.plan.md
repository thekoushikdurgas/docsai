# 1.x User Billing Era Overhaul

## Current problems (same pattern as 0.x)

Every `version_1.N.md` has these defects:

1. **Generic "Email orchestration" runtime flow** ‚Äî `v1.1` (Bulk CSV + billing) shows the same flow as `v1.0` and `v1.4`. Each minor's runtime focus must be unique and anchored to the roadmap stage it covers.
2. **Boilerplate task track bullets** ‚Äî rotating "bulk finder pipeline / P0 bug burn-down / CSV queue correctness / status truth table freeze / credit contract lock" as filler across all services identically.
3. **Missing sections** ‚Äî Backend API scope, DB/lineage scope, Frontend UX scope, UI Elements checklist, Flow/Graph delta, Cross-service ownership, Audit/Compliance notes, Patch ladder, Release Gate + Evidence sub-sections.
4. `**emailapis-user-billing-credit-task-pack.md`** has a broken PowerShell template variable `$(System.Collections.Hashtable.era)` throughout; must be replaced with `1.x`.
5. `**README.md**` is 23-line stub ‚Äî needs era summary, section-template index, cross-links.
6. **No `1.x-master-checklist.md`** ‚Äî unlike `0.x`, the 1.x folder has no shared release checklist; one must be created.
