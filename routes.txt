# DocsAI Routes & API Quick Reference

Base URL: http://localhost:8000 (or your host)

## Web route prefixes (from docsai/urls.py)

/                    → core (dashboard, login, register, logout)
/docs/               → documentation app (dashboard, pages, endpoints, relationships, postman, media, operations, CRUD, redirects)
/durgasman/          → durgasman (dashboard, design, collection, import, api/*)
/analytics/          → analytics dashboard
/ai/                 → AI agent (chat, sessions, api/chat)
/codebase/           → codebase (dashboard, scan, analyses/*)
/tasks/              → tasks (list, create, detail, start, complete, edit)
/media/              → media app (list)
/graph/              → graph visualization
/tests/              → test runner dashboard
/accessibility/      → accessibility dashboard
/roadmap/            → roadmap dashboard
/postman/            → postman app (homepage, dashboard)
/templates/         → templates list
/architecture/       → architecture (blueprint)
/database/           → database schema view
/json-store/         → json store list
/operations/         → operations dashboard
/page-builder/       → page builder editor
/knowledge/          → knowledge (list, create, detail, edit, delete, search)
/admin/              → app admin (users, logs, statistics, settings)
/api/v1/             → Documentation REST API (health, pages, endpoints, relationships, postman, dashboard)
/api/schema/         → OpenAPI schema
/api/docs/           → Swagger UI (custom)
/api/swagger/        → Spectacular Swagger
/api/redoc/          → ReDoc
/durgasflow/         → durgasflow (workflows, executions, credentials, templates, webhook)

## API endpoints (summary)

Full list: docs/api.md

- REST API: GET /api/v1/health/, /api/v1/pages/, /api/v1/endpoints/, /api/v1/relationships/, /api/v1/postman/, /api/v1/dashboard/*
- Docs AJAX: /docs/api/endpoints/draft/, /docs/api/pages/draft/, /docs/api/relationships/*, /docs/api/media-manager/*, /docs/api/media/*, /docs/api/operations/*
- Durgasman: /durgasman/api/collections/, .../requests/, .../execute/, etc.
- AI: POST /ai/api/chat/
