### About Deepgram

"Power your apps with real-time speech-to-text and text-to-speech APIs powered by Deepgram's voice AI models."

### Prerequisites

- Postman Account
    
- Deepgram Account: [https://developers.deepgram.com/](https://developers.deepgram.com/)
    

### Usage

1. Create a fork
    
2. Update collection variables
    
3. Send requests
