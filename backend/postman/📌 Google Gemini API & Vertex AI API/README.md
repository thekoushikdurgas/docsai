### About

Gemini: "Build with Gemini 2.0 Flash, 2.0 Pro, and Gemma using the Gemini API and Google AI Studio."

Vertex AI: "Google Vertex AI is a fully-managed, unified AI development platform for building and using generative AI."

### Prerequisites

- Postman Account
    
- Google Account (Gemini): [https://aistudio.google.com/](https://aistudio.google.com/)
    
- Google Cloud Account (Vertex AI): [https://cloud.google.com/](https://cloud.google.com/)
    

### Usage

1. Create a fork
    
2. Update collection variables
    
3. Send requests
