### About

"A unified interface for LLMs."

Supports auto routing to different providers. Model providers include OpenAI, Amazon, Anthropic, Google, Mistral, xAI, and more.

### Prerequisites

- Postman Account
    
- OpenRouter Account: [https://openrouter.ai/](https://openrouter.ai/)
    

### Usage

1. Create a fork
    
2. Update collection variables
    
3. Send requests
