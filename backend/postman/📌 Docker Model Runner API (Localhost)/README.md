### About

"Docker is a tool that helps developers build, share, and run applications using containers."

Docker Model Runner is a Docker Desktop plugin to run LLMs locally. It can be used inside a container or on the host via REST API or CLI.

### Prerequisites

- Postman Account
    
- Docker Account: [https://hub.docker.com/](https://hub.docker.com/)
    

### Usage

1. Create a fork
    
2. docker desktop enable model-runner --tcp 12434
    
3. docker model pull ai/gemma3
    
4. Send requests
