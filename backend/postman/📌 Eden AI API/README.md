### About

"The full stack AI platform for developers."

Supports parallel requests to multiple providers. Model providers include OpenAI, Amazon, Anthropic, Google, Mistral, and xAI.

### Prerequisites

- Postman Account
    
- Eden AI Account: [https://app.edenai.run/](https://app.edenai.run/)
    

### Usage

1. Create a fork
    
2. Update collection variables
    
3. Send requests
