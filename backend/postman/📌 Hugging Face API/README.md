### About Hugging Face

"The AI community building the future."

### Prerequisites

- Postman Account
    
- Hugging Face Account: [https://huggingface.co/](https://huggingface.co/)
    

### Usage

1. Create a fork
    
2. Update collection variables
    
3. Send requests (on 503 due to model loading, wait and retry)
