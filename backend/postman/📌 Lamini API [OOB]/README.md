### About

"Lamini is the enterprise LLM platform for existing software teams to quickly develop and control their own LLMs."

### Prerequisites

- Postman Account
    
- Lamini Account: [https://app.lamini.ai/](https://app.lamini.ai/)
    

### Usage

1. Create a fork
    
2. Update collection variables
    
3. Send requests
