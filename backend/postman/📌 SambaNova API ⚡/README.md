### About

"SambaNova Cloud is available for developers to easily use the best open source models with the fastest inference speeds."

### Prerequisites

- Postman Account
    
- SambaNova Account: [https://cloud.sambanova.ai/](https://cloud.sambanova.ai/)
    

### Usage

1. Create a fork
    
2. Update collection variables
    
3. Send requests
