# Version 10.3 - Template Forge

Focus: template authoring, preview, storage lifecycle, and variable validation.

## Patch checklist (`10.3.x`)
- `10.3.0` freeze template CRUD + preview contract.
- `10.3.1` implement template API and S3 storage adapter.
- `10.3.2` persist template metadata and hash lineage.
- `10.3.3` ship editor UI with variable insert and preview.
- `10.3.4` define render flow and safe fallback graph.
- `10.3.5` enforce unsubscribe footer and policy placeholders.
- `10.3.6` add retry handling for template fetch/render failure.
- `10.3.7` audit template versions used by campaign sends.
- `10.3.8` add template cache hit-rate monitoring.
- `10.3.9` release checklist with sample render snapshots.
- **Patch closure:** Every codenamed patch file includes **Micro-gate** + **Service task slices**. Era hub: [`versions.md`](../versions.md).
